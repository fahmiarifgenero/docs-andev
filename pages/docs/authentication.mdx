## System Authentication

Sistem autentikasi yang digunakan cukup sederhana, yaitu menggunakan `Auth::routes()` yang disediakan oleh Laravel. Untuk mengakses halaman `/dashboard` harus login terlebih dahulu. Jika belum login maka akan diarahkan ke halaman login. Jika sudah login maka akan diarahkan ke halaman `/dashboard`.
Password yang dienkripsi secara sederhana dengan menggunakan `md5` agar jika terjadi lupa password pada user dapat dibuka dari database, karena aplikasi berjalan pada server lokal, aplikasi ini tidak menggunakan faktor 2FA ataupun fitur lupa password. <mark>Jika user lupa password maka harus menghubungi admin untuk mereset akun</mark>

Secara default validasi user yang berhasil registrasi dapat mengakses halaman `/registered`. Berikut database `user` di dalam `users`.

## Account authentication

User tidak dapat melakukan registrasi secara langsung, user harus menghubungi admin untuk melakukan registrasi.
List data akun yang dibutuhkan untuk registrasi adalah sebagai berikut:

- Nama
- NIK (Nomor Induk Karyawan)
- email
- password
